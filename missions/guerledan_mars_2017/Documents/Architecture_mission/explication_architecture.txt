// niveau 1

Le mode ACTIVE correspond au lancement de la mission MOOS.

// niveau 2

Le mode DEPART correspond au commencement du levé bathymétrique.
Le mode est activé lorsque l'on atteint la position initiale du début de levé.

Lorsque la mission est terminée, le mode LOG est activé, afin de pouvoir logger les données de toute la mission.

// niveau 3

Le mode LEVE est activé lorsque l'on commence un nouveau levé (un nouveau "pattern").

Le mode QUALITY_CHECK est activé lorsque l'on a terminé un pattern.

Le mode CORRECTION est activé lorsque l'algorithme de Quality Check a indiqué qu'il était nécessaire de refaire le pattern.

La mission MOOS va ainsi effectué le levé K tant que l'algorithme de QC ne l'a pas validé, puis va passé au levé K+1 (nouveau pattern).
